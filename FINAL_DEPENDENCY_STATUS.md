## Проблемы с установкой

После установки Rust через https://rustup.rs/, система может не видеть его в PATH в текущей сессии терминала. Для решения этой проблемы:

1. Перезапустите терминал/командную строку
2. Или выполните в новой сессии терминала: `rustc --version` и `cargo --version`, чтобы убедиться, что Rust установлен и доступен

После этого можно будет установить оставшиеся пакеты:

```bash
cd remnashop
pip install aiogram~=3.2.0
pip install aiogram-dialog~=2.4.0
pip install remnawave>=2.2.6
```

## Проблема с asyncpg

Пакет asyncpg~=0.30.0 не удалось установить из-за проблем с компиляцией под Python 3.14. Возможные решения:

1. Использовать альтернативный драйвер: `pip install psycopg2-binary`
2. Попробовать использовать Docker-контейнер с более старой версией Python
3. Попробовать более новую версию asyncpg, если она будет поддерживать Python 3.14
