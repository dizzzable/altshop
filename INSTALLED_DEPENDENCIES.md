## Пакеты, которые не удалось установить

### aiogram и aiogram-dialog

Эти пакеты требуют pydantic-core, который требует Rust для компиляции. Для установки этих пакетов необходимо:

1. Установить Rust: https://rustup.rs/
2. После установки Rust выполнить: `pip install aiogram~=3.22.0 aiogram-dialog~=2.4.0`

### asyncpg

Пакет asyncpg требует компиляции с помощью Visual Studio, но не может быть скомпилирован с Python 3.14 из-за ошибок в исходном коде. Возможные решения:

1. Попробовать использовать psycopg2-binary вместо asyncpg: `pip install psycopg2-binary`
2. Попробовать установить более новую версию asyncpg, если она поддерживает Python 3.14
3. Использовать другую версию Python (например, 3.12 или 3.13), если компиляция не удается

### remnawave

Этот пакет требует Rust для компиляции. Для установки необходимо:

1. Установить Rust: https://rustup.rs/
2. После установки Rust выполнить: `pip install remnawave>=2.2.6`

## Заключение

Большинство зависимостей были успешно установлены. Некоторые пакеты требуют дополнительных шагов для установки из-за необходимости компиляции Rust-кода или проблем с совместимостью компилятора. Рекомендуется установить Rust и выполнить установку оставшихся пакетов вручную.
